# Formula Builder

Minecraft MOD


## 追加するアイテム

### 数式ビルダー

入力した数式の条件を満たす座標にブロックを設置します。

設置するブロックは数式ビルダーの上に置かれたチェストなどのインベントリを持つブロックから引き出されます。

#### レシピ

```
■B■
D|D
OOO
```

- B: 本
- D: ダイアモンド
- |: ビーコン
- O: 黒曜石
